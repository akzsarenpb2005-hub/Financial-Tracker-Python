# Financial-Tracker-Python

Personal Finance Tracker & Currency Converter
A Python-based financial tool designed to track expenses, categorize spending, and automatically convert multiple currencies into LKR (Sri Lankan Rupees) using live or historical exchange rates.

🚀 Features

Multi-Currency Support: Automatically detects currency codes (like USD, EUR, INR) in your expenses and converts them to LKR.

Smart Categorization: Groups expenses into "Important" and "Non-important" to help you identify unnecessary spending.

Budget Analysis: Compares your total spending against your monthly revenue to calculate profit or loss.

Visual Insights: Generates a Pie Chart using Matplotlib to give a clear visual representation of your spending habits.

Average Calculation: Uses NumPy to calculate your average monthly expenditure.

🛠️ Tech Stack

Language: Python

Data Handling: Pandas, NumPy

Visualization: Matplotlib

📁 Project Structure

Financial Tracker.py – The main Python logic.

Financial Tracker.csv – Dataset containing dates, descriptions, categories, and amounts.

Live Exchange Rates.csv – Reference file for currency conversion rates.
